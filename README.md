# 🎬 Movie Industry Data Analysis

## 📌 Project Overview

This project explores the movie industry using data analytics to identify the key factors that contribute to commercial success. By collecting, cleaning, and analyzing data from multiple public sources, the project provides data-driven insights into movie budgets, genres, release timing, ratings, cast, and studios.

The objective is to demonstrate how data analysis can support strategic business decisions for a company planning to enter the film industry.

---

## 🎯 Business Objective

A company planning to launch a movie production studio wants to understand what drives success in the film industry. This analysis answers important business questions using historical movie data and provides recommendations based on profitability, audience trends, and industry performance.

## 📊 Business Questions Explored

This project investigates the following questions:

1. What production budget leads to the highest profitability?
2. Which movie genres generate the greatest financial returns?
3. What is the best time of the year to release a movie?
4. Which actors and directors consistently contribute to successful films?
5. How much should a studio invest to improve its chances of winning an Oscar?
6. Does movie runtime or age rating significantly affect profitability?
7. What profit margin should a successful studio aim to achieve?
8. Which major studios represent the best benchmarks for business success?

## 📈 Key Insights

* Mid-to-high budget movies generally produced stronger financial returns.
* Animation, Adventure, and Sci-Fi were among the most profitable genres.
* Summer releases showed consistently strong performance across multiple genres.
* Certain actors and directors demonstrated significantly higher average profitability.
* Movie ratings influenced profitability depending on genre and target audience.
* Disney and Warner Bros. served as valuable benchmarks for successful studio performance.
